## Overview

`contract unpack` is a command that unpacks dotcontract file into a dotcontract directory.

## Synopsis

```
contract pack
```

## Description


## Options

```flags
-f
```
Specifies the dotcontract file to use as input.

```flags
-o
```
Specifies an output path for the unpacked dotcontract directory.
